# Current Task
Create a feature that allows the user to visually clip the model via the user of two horizontal planes. The model is only visible between the planes.

## User Interface
Below the Layer Panel on the left side of the screen, display a vertical slider control. The slider control has two bars that the user can drag up and down. The height of the slider control represents the full Z height of the print volume as set in the application settings. The user can drag the top bar down to set the upper clipping plane and drag the bottom bar up to set the lower clipping plane. The area between the two bars represents the visible area of the model.

If the upper bar is dragged below the lower bar, the lower bar moves down with it. If the lower bar is dragged above the upper bar, the upper bar moves up with it.
