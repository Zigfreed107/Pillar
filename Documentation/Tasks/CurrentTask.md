# Current Task

Use this file as a lightweight scratchpad for the active implementation task when a shared written note is helpful.

## Purpose

This file is intentionally temporary. It should contain only current planning notes that are useful for the active task, and it can be cleared or replaced as the work changes.

## Template

### Goal

Describe the user-visible outcome we are implementing.

### Constraints

List any architectural, UX, or performance constraints that matter for this task.

### Files Likely To Change

List the main code files expected to change.

### Open Questions

Capture only unresolved questions that still affect implementation.

### Notes

Add short bullets or numbered steps as needed. Archive or delete stale notes rather than letting this file become long-lived documentation.
